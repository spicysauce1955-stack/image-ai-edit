# AR Tech Survey — Web & Mobile

**Started:** 2026-05-18
**Scope:** Packages, libraries, SDKs, methodologies, and tech stacks for building AR experiences targeting browsers and phone devices.

## Layout

- `process-log.md` — chronological log of what I searched, why, what I found, decisions made
- `01-web-ar/` — WebXR, AR.js, A-Frame, MindAR, 8thWall, Niantic Lightship Web, Zappar, model-viewer, Three.js / Babylon.js + WebXR, Needle Engine
- `02-mobile-ar/` — ARKit (iOS), ARCore (Android), RealityKit, visionOS, Sceneform/SceneView
- `03-cross-platform/` — Unity AR Foundation, Unreal AR, Flutter AR, React Native (ViroReact, react-native-arkit), Expo, Capacitor approaches
- `04-methodologies/` — SLAM, marker/markerless tracking, plane detection, image/face/body/object tracking, depth & occlusion, lighting, anchors, VPS/cloud anchors, Gaussian splatting / NeRFs in AR
- `05-tech-stack/` — 3D formats (glTF/USDZ), authoring tools, AI/CV models (MediaPipe, ONNX, segmentation), backends, asset pipelines
- `06-synthesis/` — index, comparison matrix, recommended stacks per use case

## Companion
